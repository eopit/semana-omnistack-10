# semana-omnistack-10
Omnistack week 10 - DevTinder

## Commands to run:
* **yarn start** => web folder
* **yarn dev** => backend folder
  


## Routes
* GET on => /devs --> list all devs with JSON
* POST on => /devs --> register a new DEV with parameters below:
```
{
	"github_username":
	"techs":
	"latitude":
	"longitude":
}
```
